# Buttons

Buttons are used to trigger actions within an application.

## Usage

Buttons are used to trigger internal actions within your web applications.

## Buttons with no baseColor passed

When no baseColor is passed, a default baseColor from theme is used

<Preview className="abhi-preview-class" withToolbar withSource="open">
  <Story name="Basic button" >
    <Button onClick={action('button-click')}>Default baseColor</Button>
  </Story>
</Preview>

## Buttons with baseColor passed

When a baseColor is passed, the component creates a swatch based on the baseColor and uses it for styling the button

<Preview withToolbar>
  <Story name="Basic button with baseColor">
    <Button onClick={action('button-click')} baseColor="hsl(100, 50%, 50%)">With baseColor</Button>
  </Story>
</Preview>

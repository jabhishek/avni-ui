# Buttons

Buttons are used to trigger actions within an application.

## Buttons with no baseColor passed

When no baseColor is passed, the primary color (swatch/standalone color) from theme is used

<Preview withSource="open">
  <Story name="Default button" >
    <Button onClick={action('button-click')}>Default</Button>
  </Story>
</Preview>

## Buttons with baseColor passed

When a baseColor is passed, the component creates a swatch based on the baseColor and uses it for styling the button

<Preview withSource="open">
  <Story name="With baseColor">
    <Button onClick={action('button-click')} baseColor="hsl(100, 50%, 50%)">With base color</Button>
  </Story>
</Preview>
